# Projet Soutenance

Site statique. Instructions pour démarrer localement :

1. Se placer dans le dossier du projet :

```bash
cd /home/usali/Documents/projet-soutenance
```

2. Démarrer le serveur (via npm script) :

```bash
npm run start
```

3. Ouvrir le site dans le navigateur :

```bash
npm run open
# ou ouvrir manuellement http://localhost:8080
```

4. Arrêter le serveur :

Dans le terminal où le serveur tourne, appuyer sur `Ctrl+C`.

Alternative : installer et utiliser l'extension Live Server de VS Code.
